# The Future-Ready Entrepreneurship Model (FREM)

PhD Proposal Presentation - Developing and Validating a Dynamic Capabilities Framework for Navigating Future Uncertainty

## Overview

This repository contains an interactive presentation for the PhD proposal on **The Future-Ready Entrepreneurship Model (FREM)**. The presentation is built as a single-page HTML application with bilingual support (English/Persian).

## Features

- **Interactive Slides**: Navigate through 24 slides using keyboard or mouse
- **Bilingual Support**: Switch between English and Persian languages
- **Clickable Progress Bar**: Click anywhere on the progress bar to jump to a specific slide
- **Tooltip Preview**: Hover over the progress bar to see slide numbers
- **Smooth Animations**: Professional slide transitions and fade-in effects
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Usage

Simply open `phd.html` in a web browser. No server or installation required.

### Navigation

- **Next/Previous Buttons**: Click the arrow buttons to navigate
- **Progress Bar**: Click anywhere on the progress bar to jump to a specific slide
- **Language Toggle**: Click the language button to switch between English and Persian
- **Keyboard**: Use arrow keys (if implemented) for navigation

## Structure

The presentation covers:

1. Title and Introduction
2. Problem Statement (VUCA Environment)
3. Research Gap and Questions
4. Proposed Solution (FREM Model)
5. Research Methodology
6. Seven Core Pillars of FREM
7. Key Differentiators
8. Research Objectives and Questions
9. Hypotheses
10. Expected Contributions
11. Timeline
12. Limitations
13. Expected Results
14. Future Applications (Database & LLM)
15. Conclusion
